Self Made Store plugin let the users post their products on the site & show most recent product easily via widget.

== Description ==
If you activate the Self Made Store plugin after that you have daactivated it. Make sure you enable the "Show Product Inventory" checkbox in Settings -> Self Made Store Settings page. After saving it, you can, of course, disable it if needed.

Self Made Store plugin let the users to post their Products and plugin includes Widget which show the product information super easily. Thumbnail, Title, Price, Weight, Inventory. User is able to change the widget's displayed title, default is Products. And user is able to change how many product the widget would show.(Default is 3 most recent products).

Additional plugin features
* Bootstrap 3.3.6
* Widget is fully responsive
* Thumbnails is 100x100


== Installation ==
1. Upload 'plugin-directory' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Shortcodes : [smstore show=sku] , [smstore show=price], [smstore show=weight], [smstore show=color], [smstore show=inventory]
